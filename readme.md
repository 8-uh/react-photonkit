# react-photonkit

[Photon][photon] components built with React

> Developing...


## Install

```sh
$ npm install
```


## Build

```sh
$ npm run build
```

See `./dist/` directory


## Example

```sh
$ npm start
```

## test

```sh
$ npm test
```
