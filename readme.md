# [react-photonkit]

[Photon][photon] components built with React


## Install

```sh
$ npm install
```


## Build

```sh
$ npm run build
```

See `./dist/` directory


## Example with Electron

See https://github.com/react-photonkit/epp


## Test

```sh
$ npm test
```
